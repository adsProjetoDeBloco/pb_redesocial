# REDESOCIAL API

## Relações de dados

### Entidades:

1 - User:
* Id
* Nome
* Biografia
* Email
* Uma lista de jogos
* Uma lista de Seguidores
* Uma Lista de Grupos

2 - Follower:
* Id
* Lista de Usuarios

3 - Group:
* Id
* Nome do Grupo
* Descrição
* Membro
* Data da Criação

4 - Game:
*Id
*Nome do Jogo
*Categoria
*Descrição

5 - Post:
* Id
* Titulo
* Texto
* Nome do usuarios que postou
* Lista de Comentarios

6 - Comments:
* Id
* Mensagen
* Nome do usuario que publicou
* data da postagem
